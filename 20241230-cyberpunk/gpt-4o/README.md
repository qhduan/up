# 赛博朋克世界观

欢迎来到赛博朋克世界！在这个世界中，科技与人类的生活紧密相连，城市高楼林立，霓虹灯闪烁，黑客与企业巨头在虚拟与现实中交锋。

## 世界规则

1. **科技与人性**：科技高度发达，但人性与伦理问题日益突出。
2. **城市化**：城市是世界的中心，乡村几乎被遗忘。
3. **企业统治**：大型企业掌控着经济与政治。
4. **黑客文化**：黑客是新时代的反叛者，挑战权威。
5. **虚拟现实**：虚拟世界与现实世界的界限模糊。
